# Feature Proposals

Bold features to evolve RepoHub from a repo dashboard into an **AI-native dev cockpit** — the single surface for managing projects, dispatching agents, exploring packages, and previewing work.

**Direction**: RepoHub becomes the command center for a solo developer who thinks in terms of intent and agents, not windows and terminals.
